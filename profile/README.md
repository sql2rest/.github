# SQL2REST

**Read-only REST API for the JTL-Wawi.** Turn your JTL-Wawi database into a clean, documented REST API in minutes. No cloud, no official API tier, runs on your own server.

🌐 **[sql2rest.com](https://sql2rest.com)** · 📚 [Docs](https://sql2rest.com/docs/) · 🎥 [YouTube](https://www.youtube.com/@sql2rest) · 💼 [LinkedIn](https://www.linkedin.com/company/sql2rest/)

---

## What it does

An **auto-discovery wizard** detects your JTL version automatically and exposes ready-made REST endpoints for products, customers, orders, suppliers, stock and more. SQL2REST runs as a **Windows service** directly on your server, so all data stays in-house and GDPR-compliant by design.

- **Read-only** by design — no write access, no risk to your JTL-Wawi database
- **Works with every JTL-Wawi edition**, including the free Start version
- **OpenAPI / Swagger** documentation out of the box
- **HTTP-MCP endpoint** — connect Claude Desktop, Cursor, Windsurf and other AI clients directly to your live JTL data
- No-code ready: **n8n, Make, Zapier**; BI-ready: **Power BI, Tableau**
